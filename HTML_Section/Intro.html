HTML is the basic programming Language for Front-End Development.

This language is easy to learn with writting.

The HTML is stands for 'Hyper Text Markup Language'.

<!DOCTYPE html>:

This tag defines that this document is an HTML5 document.

<html>:

It defines that where the HTML Coding is start.

<body>:

Under this tag all the coding's content is print in web page.

<p> , <h1>:

These two tags will discuss in the Element topic.

</html> , </body>:

These two tags used for close the body and html element.

Note: HTML tag open with <> and close with</>

Make your own First Website using HTML

Open your Notepad

Create a new file and type as shown in the box

Save your file as "My first website.html"

Re-open the file, Your first Website is appear.

Your First Website

<!DOCTYPE html>

<html>

<body>

<h1>This is my first Website </h1>

<p>This is my first Website </p>

</body>

</html>

Try it with yourself

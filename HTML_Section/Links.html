We can connect one file to other one or more files using link tag.

Learn about how to insert an link tag.

And step by step process.

Inserting a link

To insert a link make sure you have at least one file for link.

In the second file you can write or insert an image and you can add your file name for link.

In the first file where you link the second file, name 'Links.html'.

See the Example box below:

Example

<!DOCTYPE html>

<html>

<body>

<h1>HTML Links</h1>

<p>This is a <a href="2nd HTML.html">link.</a></p>

</body>

</html>

Output

Explanation:

a : It stands for anchor tag.

href : It is used for typing the file name under quotes will be linked.

Under the anchor tag you will write any text, the text will be converted into a link

note: It is compulsory to write a word or letter to make it visible link.

You can add more links like that but you have to create more files for more links.

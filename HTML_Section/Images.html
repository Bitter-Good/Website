Inserting an image

First make a source for image or download an image for inserting the image in your website.

Note: The file is created in the folder then image will be also created in that of folder.

To inserting an image see the Example box.

Example

<!DOCTYPE html>

<html>

<body>

<img src="logo.png" alt"image">

</body>

</html>

Output

Explain

img It stands for "image" to start the image tag.

src It stands for "source" it means you have to type your image name.

alt It is used for name in the website and it helps when the website is not fully reloaded.

The Image Tag is also an End Tag.

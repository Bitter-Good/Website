What is Element?

An HTML element is everything from the start tag and end tag

Some of Examples are here:

Example

<p>This is a Paragraph </p>

<h1>This is a Heading </h1>

Try it with yourself

Note: Some HTML elements have no content (like the <br> element). These elements are called empty elements. Empty elements do not have an end tag!

Nested Elements

HTML elements can be nested (this means that elements can contain other elements).

All HTML documents consist of nested HTML elements.

The following example contains four HTML elements (<html>, <body>, <h1> and <p>)

Example

<!DOCTYPE html>

<html>

<body>

<p>My first paragraph.</p>

<h1>My first Heading.</h1>

</body>

</html>

Try it with yourself

Paragraphs

The paragraph tag is used to write the words in paragraph in html.

It is an easy concept to learn about paragraphs.

The paragraph tag has a syntax(this means language of coding) that we use to write paragraph in html.

Paragraph also a Nested Element.

It syntax is:

<p>This is a Paragraph </p>

Heading

The heading tag is used for making header for webpage.

The heading tag has also a syntax to write a heading.

The heading tag is also a Nested Element.

It Syntax is:

<h1>This is a Heading </h1>

Heading has six Differnt sizes

Example

<!DOCTYPE html>

<html>

<body>

<h1>My first Heading.</h1>

<h2>My second Heading.</h2>

<h3>My third Heading.</h3>

<h4>My fourth Heading.</h4>

<h5>My fifth Heading.</h5>

<h6>My sixth Heading.</h6>

</body>

</html>

Try it with yourself

Bold

The Bold tag is used to bold a word or sentence or letter.

It is also a Nested Element.

How to use it, see in the Example Box:

Example

<!DOCTYPE html>

<html>

<body>

<p>I am Normal.</p>

<p><b>I am Bold.</b></p>

</body>

</html>

Try it with yourself

It is also a Nested Element.

How to use it, see in the Example Box:

Example

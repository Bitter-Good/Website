HTML Attributes are placed in Elements.

Attributes are used for Styling an Element.

We will learn the Attributes in detail.

What is Attributes?

The Attributes is used for Style a single or particular Element.

In the bottom you can see the code format for using Attributes.

Example

<!DOCTYPE html>

<html>

<body>

<p style="font-size: 30px;">I am Big Paragraph with 30px.</p>

<p style="color: red;">I am colored Paragraph with Red colour.</p>

</body>

</html>

Try it with yourself

  • Attributes are Works in an End and Nested Tags.

Using "Style" Attribute.

This style code is used for starting the Attribute Tag or Styling an Element.

It is a way for using it you have to take a space between an element and bracket.

You can type style to start designing for an Element.

Colouring an Element using Attribute.

In the Example box below.

Example

<!DOCTYPE html>

<html>

<body>

<p style="color: red;">I am colored Paragraph with Red colour.</p>

<p style="color: green;">I am colored Paragraph with Green colour.</p>

<p style="color: blue;">I am colored Paragraph with Blue colour.</p>

</body>

</html>

Try It with yourself

Note: Do not Type "colour" instead of "color" it will through error and do not change the colour.

Changing an Element's size using Attribute.

In the Example box below.

Example

<!DOCTYPE html>

<html>

<body>

<p style="font-size: 30px;">I am 30px big.</p>

<p style="font-size: 40px;">I am 40px big.</p>

<p style="font-size: 50px;">I am 50px big.</p>

</body>

</html>

Output

Note: Do not Type "colour" instead of "color" it will thow error and do not change the colour.

  • You can add infinite size you want!

What is Margin?

A margin is a gaping attribute that can makes gap from outside.

You can add background-color Property to check the margin.

Example

<!DOCTYPE html>

<html>

<body>

<p style="margin: 10px; background-color: yellow;">I am gaped by Margin 10px.</p>

<p style="margin: 20px; background-color: yellow;">I am gaped by Margin 20px.</p>

<p style="margin: 30px; background-color: yellow;">I am gaped by Margin 30px.</p>

</body>

</html>

Output

  • The distance between the Paragraphs are "Margins".

  • The Margin can be Adjust from four side.

  • Margin is a Gap that can be Adjust from Outside.

  • These are series of Margin: Top, left, Bottom, Right.

What is Padding?

A padding is a gaping attribute that can makes gap from inside.

Example

<!DOCTYPE html>

<html>

<body>

<p style="padding: 10px; margin: 5px; background-color: yellow;">I am gaped by Padding 10px.</p>

<p style="padding: 20px; margin: 5px; background-color: yellow;">I am gaped by Padding 20px.</p>

<p style="padding: 30px; margin: 5px; background-color: yellow;">I am gaped by Padding 30px.</p>

</body>

</html>

Output

  • In the Output, the thickness of yellow color will be increased.

  • This will be show the Padding.

  • We have also add the margin to see the diffrent Paragraphs padding.

  • Padding is a Gap that can be Adjust from Inside.

  • Padding have also series like Margin but this topic will be cleared later.
